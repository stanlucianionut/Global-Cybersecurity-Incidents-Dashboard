## **Overview**

This project provides a fully interactive **Power BI dashboard** visualizing global cybersecurity incidents from 2015–2024.  
It tracks incident counts, financial losses, affected users, threat actors, and attack types with rich drilldowns and cross-filtering.

All visuals update dynamically based on selections, enabling deep and intuitive data exploration.

---

## **Key Features**

### **1. Dynamic KPI – Total Incident Count**  
Automatically updates based on slicer selections (country, year, attack type, threat actor).

### **2. Yearly Incident Trend**  
Line chart showing incident frequency from 2015–2024.

### **3. Financial Loss Analysis**  
Top-N bar chart highlighting the most financially damaging incidents across countries or industries.

### **4. Affected Users by Attack Type (Drilldown)**  
Multi-level hierarchy:  
- Attack type  
- Vulnerability exploited  
- Defense mechanism used  

### **5. Threat Actor Breakdown**  
Donut chart showing distribution of nation-states, insiders, organized groups, and unknown actors.

### **6. Full Cross-Filtering & Slicers**  
Interactive filtering across:  
- Country  
- Year  
- Attack Type  
- Attack Source  

All visuals update instantly.

---

## **Dataset**

**Source:** Kaggle – Global Cybersecurity Incidents  
Includes attributes for:  
- Attack types  
- Target industries  
- Threat actors  
- Financial loss  
- Affected users  
- Vulnerabilities  
- Defensive actions  
- Country & time metadata

## **Technologies Used**

- Power BI Desktop  
- DAX Measures  
- Data Modeling  
- Hierarchies & Drilldowns  
- Visual Interactions / Cross-filtering  
- Top-N Logic  
- Dashboard UI/UX Design  

---

## **How to Use**

1. Download or clone the repository.  
2. Open the `.pbix` file in **Power BI Desktop**.  
3. Interact with slicers or any visual to filter the dataset.  
4. All charts will update automatically.
